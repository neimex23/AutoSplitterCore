AutoSplitterCore Extension for HCM

This is an extension for HitCounterManager that allows you to automatically split when an event happens in a game.

Features

  • AutoSplit for:
    • Dark Souls 1
    • Dark Souls 2
    • Dark Souls 3
    • Sekiro: Shadows Die Twice
    • Elden Ring
    • Hollow Knight
    • Celeste
    • Cuphead
    • Dishonored
  • Auto-start timer when the game run is started.
  • Use In-Game Time or Real-Time values on the internal timer, delta, total time, and PB in OBS layout.
  • Save file for all settings of AutoSplit: HitCounterManagerSaveAutoSplitter.xml (compatible with future versions).

AutoSplitter Flags

  • Sekiro:
    • Kill a Boss
    • Activate an Idol
    • Trigger a position
    • Custom flag
  • Elden Ring:
    • Kill a Boss
    • Activate a Grace
    • Trigger a position
  • Dark Souls 1:
    • Kill a Boss
    • Activate a Bonfire
    • Level up the character
    • Trigger a position
    • Obtain an item

AutoMods Features

  • Sekiro: No Logos Intro, No Tutorials
  • Elden Ring: No Logos Intro

Requirements

  • Software: .NET Framework 4.8.1 or newer.
  • OS: Windows (64-bit only).
  • Execute the program as Administrator.

Installation

  • Portable:
    • Decompress the .zip file "AutoSplitterCore_Portable_v3.x.0" into a folder and copy the content to the installation directory of HitCounterManager.
  • Installer:
    • Install "AutoSplitterCore_Installer_v3.X.0.msi" in the same folder as HitCounterManager.
This uninstalls the previous version automatically and installs the new version. You must have installed the MSI before updating.
